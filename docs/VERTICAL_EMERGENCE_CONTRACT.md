## 0. なぜ縦の軸が要るか（既存3軸では測れない一次証拠）

現在のガバナンスは、一つの主張を**3つの直交する軸**で監査している：

| 既存軸 | ドキュメント | 何を測るか | 方向 |
|---|---|---|---|
| **因果的閉包 CC-Level (C0–C4)** | `CAUSAL_CLOSURE.md` | 主張の因果生成を**準備過程のどこまで遡ったか** | 時間・上流（provenance） |
| **創発 Level (L0–L8)** | `EMERGENCE_LEVELS.md` | 一つの run が t=0 から**どこまで中断なく育ったか** | 時間・前進（depth） |
| **証拠検証 evidence_validation** | `EVIDENCE_CONTRACT_V2.md` §1.1 | その主張が**どの次元で測定検証されたか** | 次元（2D/3D） |

**この3つのどれも、「記述のスケールをまたぐ」軸ではない。** 現実の宇宙では、流体・細胞・脳は基礎物理と
別の物質でできているのではなく、**同じ全体状態を別の尺度で粗視化**したときに別の有効法則として現れる。
Aeterna がもし将来「量子 → 古典場 → 化学 → 個体 → 適応系」と段を積むとき、既存3軸だけでは
**「複数のモデルを順番に走らせただけ」と「一つの下位物理から有効階層が形成された」を区別できない**——
これは `ANTI_DRIFT.md` が最も恐れる drift そのものである。

必要なのは、途中に完成品（有効変数・有効法則）を**手で挿入したか**、それとも**下位状態から導出されて
現れたか**を、スケール境界ごとに機械検証する監査だ。本書はその軸——**スケール橋 Level（Bridge-Level,
B0–B3）**——を前登録する。

> **一行で**：横に新しい白を増やす前に、量子↔古典場↔化学↔個体↔適応の**間の「橋」が本当に下位物理から
> 生まれたか**を測る縦方向の監査を、既存の `target_encoded` / 第8監査の規律を**スケール間へ一般化**して作る。

> **用語の規律（外部レビュー、2026-07-17）**：本書で「宇宙・生命・脳」等の強い語は、(a) **現実世界の
> 動機づけとなる既知現象**（実在の生命・脳が示す下向き因果など＝Aeterna の主張ではない）、または
> (b) Aeterna が**まだ達成していない研究の方向・候補**（「生命の器候補」「脳候補」方向）のいずれかを
> 指す。**Aeterna が生命・脳・観測者を作った・示したという主張として読んではならない**。形式的定義・
> 観測量・限界・測定基準を伴わない箇所は、すべて後者（候補・方向）の意味である
> （`EVIDENCE_CONTRACT_V2` §9.1 の禁止主張リストと同じ規律をスケール軸に適用）。

## 1. 新しい軸：スケール橋 Level（B0–B3）

一つの「橋」とは、**下位記述 D_lo（微視）と上位記述 D_hi（有効・粗視）のペア**である。橋は、下位状態を
粗視化写像 Π で潰したとき、上位に有効変数・有効法則が現れるか、を問う。**Level ではなく橋ごとに**付す。

| Level | 名前 | 意味 | 判定の骨子 |
|---|---|---|---|
| **B0** | Hand-stacked | 上位変数・上位法則を**人が直接与えた**（下位から導出していない） | ❌ 縦の創発としては**却下**（`target_encoded` 相当のスケール間版） |
| **B1** | Derived-variable | 上位変数が下位状態から**測定で定義**されている（§2 ゲートI） | 有効変数は emerged、有効法則はまだ | 
| **B2** | Closed-effective-law | 上位変数が**予測可能・再現可能・閉じた時間発展・粗視化スケールに対して頑健**（§2 ゲートII） | 有効法則が emerged |
| **B3** | Downward-realized | 上位状態の下位へのbackreactionが、**別の外力ではなく下位の局所相互作用として実現**されている（§2 ゲートIII） | 縦の因果ループが閉じた |

**CC-Level との関係（直交・独立）**：CC-Level は「準備を上流へどこまで遡ったか」、Bridge-Level は
「記述スケールを下位へどこまで根拠づけたか」。両方高いのが理想だが、片方だけ高い橋もありうる
（例：C1 で seeded した下位状態でも、そこから B2 の有効法則が導出できる——下位状態の由来と、
下位→上位の橋の健全性は別問題）。**混同しない。**

## 2. 橋契約（Bridge Contract）の三ゲート

各橋に、下から順に3ゲートを課す。上のゲートは下のゲートを前提とする（B1⊂B2⊂B3 のように積む）。

### ゲート I — 導出（Derivation）：上位変数は下位状態から測って定義されているか

上位の各有効変数 u（密度・温度・圧力・化学種濃度・境界・個体ID・記録状態…）が、**人が直接付与した
定数・ラベルではなく、下位状態 x から計算された関数 u = f(x)** であること。

- **PASS 条件**：`u` を生成するコードが下位状態 `x` のみを入力に取り、`u` の値が下位 run から一意に再現される。
- **FAIL（B0）**：`u` を IC/config で手置き、または下位状態と無関係に固定した。
- **機械検証（後続 PR で lint 化）**：これは既存の**第8監査 `target_encoded=false`**の**スケール間への一般化**
  である。第8監査は「完成構造を t=0 に置いていないか」を問う。ゲート I は「有効変数を下位状態抜きで
  与えていないか」を問う——同じ規律を、時間軸ではなくスケール軸に適用する。

### ゲート II — 有効法則（Effective-law）：粗視変数は閉じて予測でき、スケールに頑健か

導出された `u` の時間発展が、**下位を毎回解かずに `u` だけで**（近似的に閉じた形で）予測・再現でき、
**粗視化スケール（ブロックサイズ・サンプリング間隔）を変えても残る**こと。

- **PASS 条件（4点すべて・前登録の決定的閾値）**：各量は `EVIDENCE_CONTRACT_V2.md` §6 と同じく
  **由来・単位・pass/fail 境界を数値で前登録**する（後付けで緩めない）。以下は**前登録デフォルト値**であり、
  白ごとの上書きは `vertical_bridge` ブロック（§6）に理由付きで記録し、**結果を見てからの緩和は禁止**：
  1. **予測可能**：正規化予測残差
     `R_pred = ‖u_pred(t+Δ) − u_full(t+Δ)‖₂ / ‖u_full(t+Δ) − ⟨u_full⟩‖₂ < 0.10`
     （下位フル解を ground truth・時間窓と Δ は白ごとに登録）。
  2. **再現可能**：**独立 seed ≥5**で、有効法則の各係数の変動係数 `CV = σ/μ < 0.15` かつ関数形が同一
     （指数・冪などの形状パラメータが seed 間で ±0.1 または 95%CI 重複以内）。
  3. **閉性**：閉性残差 `R_close = Var(dU/dt − F(U)) / Var(dU/dt) < 0.20`
     （`F(U)` は上位変数のみの閉じたモデル。無視した下位自由度が dU/dt 分散の 20% 超を支配したら FAIL）。
  4. **スケール頑健性**：粗視化ブロック線長を**≥3 段**（最小と最大で線長比 ≥4×）変えて、有効法則の
     **関数形が不変**（形状パラメータが上記2の許容内で保たれる。係数の連続変化は可、関数形の破綻は不可）。
- **FAIL**：上記のいずれか1つでも境界を超える。特にスケールを変えると有効法則が消える/別物になる＝
  それは有効法則ではなく特定粗視化の見かけ。**preregistered（前登録）と post-hoc（結果を見て決めた）を
  必ず区別して記録**（`EVIDENCE_CONTRACT_V2` §6.6 の規律を継承）。
- **既存資産との接続**：これは `EVIDENCE_CONTRACT_V2.md` の **`reproducibility`＋`convergence`** を、
  **スケール頑健性**という新チェックと束ねたもの。§3 の保存語彙のうち
  `independent_estimator_agreement`（G002 の二 Nusselt）は、**別の導出経路が同じ有効量に一致する**
  という意味で、ゲート II の強い証拠形態にあたる。

### ゲート III — 下位実現（Downward-realized）：上位の下位への影響は「別の力」でなく局所相互作用か

生命・脳では、下から上が生まれるだけでなく、**上位状態が下位を変える**（境界が内部反応を変え、
内部反応が境界を維持し、神経活動が運動を変えて次の入力を変える）。ただし——ここが縦の創発の核心——
**「上位存在が新しい力を出す」のではなく、その下向き作用が下位の局所相互作用として実現されている**
必要がある。さもなくば、それは創発した階層ではなく、外から手で足したカップリング項である。

- **PASS 条件**：上位変数 `U` が下位量 `L` に及ぼすと主張する影響 `dL/dt` が、**モデルに既にある同じ
  微視的局所相互作用項だけ**で完全に説明できる。`U` を**外部入力として係数に持つ新項が要らない**。
- **反証テスト（必須・matched control）**：下位の微視実現を無差別にスクランブル/凍結すると、`U` とは
  無関係に効果を生む局所ダイナミクスまで壊れてしまい、効果消失が「`U` 経由の因果」を切り分けたことに
  ならない（外部レビュー、2026-07-17）。正しい対照は、**下位の関連統計量と局所ダイナミクスを保ったまま、
  提案された `U`→下位の経路だけを除去する matched control**：`U` が下位に作用すると主張する経路のみを
  遮断（例：`U` が変調すると主張する局所相互作用の変調成分だけを、その時間・空間平均値に固定して
  「揺らぎを通じた `U` 依存」を消す）し、**下位の周辺分布・素の局所更新則・エネルギー収支は不変に保つ**。
  この matched control 下で主張する下向き効果が**消え**、かつ**通常 run では残る**ときに限り、効果が
  `U` を経由していると言える（効果消失は**この matched control の下でのみ**証拠になる）。要件は変わらず：
  `U` は下位自由度**を通じてのみ**因果力を持つ（下位物理に還元されない新項を要しない）ことの実証。
- **FAIL（B0 的）**：backreaction を `if 個体ID==k: 力を加える` のような、下位物理に還元されない
  外挿ラベル操作で実装した。
- **これが最も難しく、最も重要なゲート**。B3 を通った橋だけが「下向き因果が微視物理として実現された」
  と言える。多くの既存資産は B1/B2 止まりで正常——B3 は生命・脳方向の白で初めて本格的に要求される。

## 4. 禁止主張（橋から言ってはいけないこと・`EVIDENCE_CONTRACT_V2` §9.1 の拡張）

| 禁止する言い回し | 理由 | 正しい言い回し |
|---|---|---|
| 「上位変数が下位から相関して見える＝橋が創発した」 | 相関は導出（ゲート I）ではない。手置きの `u` でも下位と相関しうる | 「`u=f(x)` として下位状態のみから計算され、独立 run で再現される」まで確認して初めて B1 |

| 「上位が下位を変えた＝下向き因果が創発」 | 手で入れたカップリング項でも下位は変わる；下位を無差別に壊しても効果は消えるが `U` 経由の切り分けにならない | ゲート III の matched control（下位統計・局所ダイナミクスを保ち `U` 経路のみ除去）下で効果が消え、通常 run では残る、を通して初めて B3 |
