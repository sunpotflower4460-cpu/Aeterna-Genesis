# WHITE_CEILINGS — 白ごとの天井地図（0 から自然な物理でどこまで登るか）

> **これは何か**：一つの「**白**」（＝固定した場の集合と法則）を選び、**初期条件だけ**を変えて t=0 から放置し、
> Level 0–8 のどこまで**自然に**登るかを測り、**届かない Level は「この白の正直な天井」**として記録した地図。
> 鉄則は [`ANTI_DRIFT.md`](ANTI_DRIFT.md)：**答えを置かない・場/項を足したら別の白・問い直しは白の捉え方へ。**
> Level 定義は [`EMERGENCE_LEVELS.md`](EMERGENCE_LEVELS.md)、成果一覧は [`PIECES.md`](PIECES.md)、
> 床は [`honest_floors.md`](honest_floors.md)。実行：`python -m ai_lab.lab --mode lawscan`。

## 天井地図（すべて t=0・IC は種/ノイズ/対称のみ・第8監査 clean・Level は measured）

| 白 (white) | 場と法則 | 到達（正直な天井） | tier | 天井の理由（なぜここで止まるか） |
|---|---|---|---|---|
| **g001 TDGL** | 複素スカラー ψ・**緩和**GL クエンチ | **L2**（位相巻き渦） | measured | 緩和勾配流＝**運動量/移流がない** → 自発運動が出ない。単一場＝繁殖用の第2自由度もない |
| **g002 Boussinesq** | 温度θ＋速度・浮力対流 | **coherent L3**（循環・KE 0→112・∮≠0） | measured | **運動量/移流がある**ので流れが出る。だが大域対流ロール＝**局在した持続的個体でない** → L4 手前 |
| **cgl** | 複素スカラー A・**振動**GL | **spiral-defect turbulence**（L1 模様＋L2 芯＋乱流運動・天井 < L4） | measured | 振動/分散で渦芯が**動く**（乱流運動・autocorr→0）が、**乱流（非 coherent）**で持続的個体を作らない。\|A\|~1 は IC 設定（**床からの成長でない**） |
| **swift_hohenberg** | スカラー u・双安定パターン形成（SH35） | **L4-static**（持続・自己修復する局在個体・自発運動なし） | measured | 局在状態は**個体**（内外差＋摂動から自己修復＝置けない）だが**変分**（勾配流）で**動かない**。自走個体（L4∧運動）は frontier |
| **gray_scott** | 2種 U,V・反応拡散 | **L7-partial**（自己複製・分裂のみ） | measured | パターンは自己コピーするが、U,V に**遺伝する内部状態の自由度がない** → 遺伝は生まれない（置けば別の白） |
| **g003 Model H** | φ＋流体・相分離×流れ | L5 co-differentiation は **frontier**（測度 WIP） | frontier | co-differentiation 測度が未収束 |

> マスクする＝TDGL 用に調整されている）。CGL の欠陥は**振幅ホール芯**で数えるのが正直。→ 測度の限界も記録。

## 天井の共通構造（なぜ止まるか — 一言で）

- **運動量/移流がない白（TDGL）** は **L2** で止まる（動けない）。
- **運動量がある白（Boussinesq）** は **L3（流れ）** まで行くが、**局在＋持続の機構**がないと **L4（個体）** に届かない。
- **振動場（CGL）** は**運動（乱流）**を出すが **coherent でなく個体化しない**（turbulent ≠ coherent）。
- **双安定パターン形成（Swift-Hohenberg）** は**持続・自己修復する局在個体（L4）**を出すが、**変分**なので**動かない**（静的個体）。
- **多種反応拡散（Gray-Scott）** は**自己複製（L7-partial）**まで行くが、**遺伝する内部自由度**がないと FULL L7 に届かない。
- **共通の理由**：ある Level に届かないのは「**その Level を可能にする自由度・機構がこの白にない**」から。
  **足せば届く。だがそれは別の白**（`ANTI_DRIFT.md` 原則1/2）。だから天井は「この白の正直な限界」。

**注目**：**同じ "L3-ish 運動" を 2 つの白が別ルートで出す**——Boussinesq は**材料の対流（coherent L3）**、
CGL は**位相欠陥の乱流運動（turbulent, 天井 < L4）**。同じ「動き」でも白が違えば質が違う（同じ数学 ≠ 同じもの）。

## 捉え方の問い直し：Level は一本のはしごではない（二軸の再枠づけ）

**地図をよく見ると、Level を「一本のはしご」で数えるのが捉え方の誤りだと分かる。** 個体性（L4）と自発運動
（L3）は**独立した二つの軸**で、単純な白は**片方を得るともう片方を失う**：

| 白 | 局在（個体化） | 自発運動 | 結果 |
|---|---|---|---|
| Boussinesq | ✕（大域ロール） | ⭕ coherent | 動くが個体でない |
| CGL | ✕（乱流） | ⭕ turbulent | 動くが個体でない |
| **Swift-Hohenberg** | **⭕（自己修復する局在個体）** | **✕（変分で静止）** | **個体だが動かない（L4-static）** |
| gray_scott | ⭕（スポット） | △（分裂） | 複製するが自走個体でない |

- **L4「個体」の判定**（`EMERGENCE_LEVELS.md`）＝ **内外差 ∧ 追跡ID持続 ∧ 摂動からの回復（自己修復）∧ サイズ非依存**。
  SH 局在状態はこれを**全て measured で満たす**（振幅~1.4 内 vs ~0 外・凍結した attractor・上半分を消しても**再成長**
  ・N=64/96/128 で同一）。**自己修復は"置けない"**——これが L2 の凍結欠陥と L4 個体を分ける決め手。
- だが SH は**変分（勾配流）**なので個体は**動かない**（重心ドリフト = 0.0）。**自走する個体（L4 ∧ 自発運動）＝
  地図の"本当の欠けている段"**は、どの単純な白も出していない。
- **誰も立てていない問い（この地図が示す）**：**局在（個体化）と自発運動を一つの白で同時に買う最小の材料は何か?**
  ＝ 非変分（時間反転を破る駆動）＋ 短距離活性化・長距離抑制の局在機構。3成分反応拡散の自走ソリトンが候補だが、
  今回の到達解像度では安定化に至らず **frontier**（原則5：避けず 0 から挑んだが、正直に天井として記録）。

## うえきの最深の問い（正直に・frontier を隠さない）

1. **白から遺伝が生まれ得るか?**
   純 RD(U,V) では**否**（天井 L7-partial）。遺伝には「**バイスタブルで複製される内部変数**」が要り、それは
   自由度を足す＝別の白。**「なぜ出ないか」＝この白に heritable な自由度がないから。** 単一根源白から遺伝が
   創発するかは **frontier（未達）**。問い直すなら「もともと内部状態を持つ白とは何か」＝**白の捉え方**へ
   （答えの配置＝タグ場 T を足す、ではない）。**注**：SH で個体（L4）は生まれたが、それは**単一の attractor**＝
   系統ごとに違い（allele）を持たない → **持続する個体があっても、まだ遺伝はない**。遺伝には「複製される個体が
   **連続的な内部状態**を真に受け継ぐ」白が要り、それを"置かず"に生む白は **frontier**。

2. **白から次元・空間が生まれ得るか?**
   固定格子の白は**空間を"置いている"**（原則1の空間版）。次元が**関係だけ**から出るのは
   **関係論/因果集合の白**：`experiments/e014`（順序 → 次元 d=2,3,4 → 1.99/2.97/3.95、分母2）、
   `e023`（順序 → 時間）、`e031`/`e032`（因果作用・地平）。＝**場-on-格子の白では次元は生まれない（置いている）。**
   次元が生まれるのは別框（causal set）で **measured**。tier=measured（因果集合系）／場系では frontier。

3. **一つの根源的な白から、どこまで生まれてくるか?**
   **最深 frontier（未証明の夢）。** 現状の正直なマップ：L2〜L7-partial の断片が**別々の白から別々に**出るが、
   **一つの白が全部を登った例はない。** 各白は「その Level 用の自由度」しか持たず、繋ぐには足す＝ drift。
   → **成果は「どこまで行けて、どこで天井になり、その天井はなぜかを正直にマップしたこと」。** 届かなくてよい。

## この地図の使い方

- 新しい白を試すとき：0 から回し、届いた Level と**天井の理由**を上表に追記。**足して天井を越えない。**
- 天井にぶつかったら：(a) なぜ出ないか測る → (b) **白の捉え方**を問い直す（原則2）→ (c) 答えを置いて回避せず
  **天井として記録**（原則4）。
- 迷ったら **partial か frontier** に落とす。合言葉：**「それは生まれたのか、置いたのか?」**
