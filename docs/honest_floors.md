# honest_floors.md — 誠実な床（誇張を止める原理的な限界）

> **これは何か**：測定で消せない・言い換えでは越えられない**原理的な床**。tier を上げる前に必ず通す関門。
> 罠（[`traps_museum.md`](traps_museum.md)）が「踏んだ失敗」なら、床は「そもそも越えられない線」。
> **床を隠すと誇張になる。** ここに書いた床は、成果（[`PIECES.md`](PIECES.md)）の上限でもある。

## 大原則

> **「同じ数学 ≠ 同じもの」** ／ **「差 ≠ 現実化」** ／ **「imposed ≠ emergent」** ／
> **「作れる ≠ 本物」（sim の underdetermination）**

---

## F-floor：「差が立つ」と「現実化する」は別

- 0≠一様から**差・模様**が立つのは measured（A2, D6, E5 等）。だが「それが**現実化**した」「**何か**に
  なった」は測定ではなく**解釈**（F1/F2 は interpretive、F3 は clip が結果を作る measured-negative）。
- **床**：actualization を measured と呼ばない。engine は「差を作る」までが測定、その先は frontier/解釈。

## S-floor：「同じ数学 ≠ 同じもの」

- メキシカンハット＝Higgs の数学、渦線 ≒ 宇宙ひも、もつれ→重力 ＝ **線形化 Einstein**、
  source-window 面積則 ≒ ホライズン。**どれも数学の一致＝analogy**であり、対象そのものではない。
- **床**：これらは tier=**analogy** に固定。「電子」「分子」「脳」「意識」「生命」「証明された」等の**生命語・
  断定語を物理主張に使わない**（LAW.md 禁止語）。

## I-floor：「imposed ≠ emergent」

- **CDT の因果性は imposed**（因果順序を手で入れて次元を測る）。B5 の diameter_dim は measured だが、
  因果性そのものは創発ではない。
- **E3 の適応度地形は imposed**（赤の女王の landscape を与えている）。turnover は measured だが開放性は frontier。
- **床**：入れたものは「入れた」と書く。imposed を emergent と混同しない。

## 第8監査 floor：完成形を始原条件に入れない

- 主張したい構造（三角形・液滴・分裂後の 2 つ・双曲の木・巻き数・√c4）を IC に埋めない。
  埋めたら measured に見えても **target_encoded**（→ T-√c4, T-dim2）。
- **床**：IC family は **種・ノイズ・対称構造・ランダム位相のみ**（white/lowk/highk/single_seed/
  sparse_seeds/ring/gradient/seeds_phase/spectral_powerlaw/bandpass）。eval ゲート・成功判定・保存則・
  監査閾値も結論を先取りしない。
- **位相制御の床（phase control ≠ target）**：サンドボックスの大量探索で「位相なしの種＝ hit 0%／
  位相を持つ family ＝~10%」が出た。**ランダム位相**（seeds_phase の一定ランダム位相・spectral/bandpass の
  ランダム位相スペクトル）は無秩序の一部で 第8監査 OK＝**位相が効く**のは本物。だが **`vortex_charges`
  （±巻き数を種に置く）は GL の target（巻き数）を埋め込む＝ target_encoded** で越えられない床。
  「位相が効く」を主張してよいのは**巻きを置かない**random-phase 系まで（→ traps_museum.md T-vortexseed）。

## Measure floor：スコア/指標は現象を測れているか

- 複雑さ窓の**正規化エントロピーは 1.0 に飽和**しやすく、ランキング信号を殺す（→ T-entropy-sat）。
- 「dim=2」は 2-cell 構成の副作用で幾何ではない（→ T-crumple2, T-dim2）。furrow は材料を削る（→ T-furrow）。
- **床**：スコアは**非飽和 measure**で、run 間に分散が残ることをテストで担保。次元主張は独立手法で二重化。

## Law-class floor：深い Level は「スコア」でなく「別の法則」から来る（turbulent≠coherent / spots≠life）

- **深い Level は法則が決める**：GL は 2D で **L2** が上限（位相巻き）。**L3 の自発循環**は Boussinesq、
  **L7 の自己複製**は Gray-Scott——**同じスコアを上げても Level は上がらない**。深くしたいなら**始原条件でなく
  法則クラスを変えて 0 から**（`ai_lab --mode lawscan`）。
- **turbulent ≠ coherent（L3 の床）**：循環があっても、KE がいつまでも揺らぐ**乱流 churn** は coherent な
  定常ロールと同じではない。L3 は flag（`coherent_rolls` / `turbulent_churn`）で区別し、churn を coherent と
  売らない（→ traps 予備軍）。
- **spots ≠ life（L7 の床）**：Gray-Scott の自己複製スポットは**反応拡散の場現象**（Pearson 1993）であって
  細胞・生命ではない。tier=measured（複製という測定量）だが、**生命語を使わない**（LAW 禁止語）。
- **遺伝は"置いた"もの＝ L7-partial の床（ANTI_DRIFT）**：Gray-Scott(U,V) の白から生まれるのは**自己複製・
  分裂**まで＝**天井は L7-partial**。**遺伝（heredity）は U,V から生まれない。** 遺伝を出すには遺伝場 `T` を
  **足す**しかなく、それは「答えを置く」（`ANTI_DRIFT.md` 原則1：買ってきたリンゴを枝にくくりつける）。
  → **「Gray-Scott が IC-only で FULL L7」は取り下げ**。T 系は `inheritance_placed=True` と正直にラベルし
  full-L7-emergence を主張しない。「白から遺伝が生まれ得るか」は **frontier**（出ないなら天井として記録）。
- **Model H の L5 は frontier**：相分離×流体の co-differentiation 測度は未収束＝frontier。screenable だが L5 を
  measured とは言わない。

## Sandbox floor：サンドボックス ≠ リポジトリの measured

- P 群（AB リング・Ricci・E-gap 等）は別チャットのサンドボックス測定で、**このリポジトリに未収録**。
- **床**：repo アンカーの無い measured は **frontier 扱い**。tier は付けてよいが、再現をこのリポジトリで
  取るまで「repo の measured」とは言わない。

## Dimension / lattice floor：空間は与えている

- ほとんどの実験は**固定格子**＝空間（次元・計量）を与えている。B5 は因果性 imposed、e017 は線形 onset。
- **床**：動的幾何（重力・時空の創発）は analogy/frontier。次元が「関係だけから出る」のは因果集合系の
  measured（B2/B3/R1）に限る。

## 床を守る運用（tier を上げる前のチェックリスト）

1. その構造を **IC に入れていないか**（第8監査）？ → 入れていれば trap。
2. それは **測定か解釈か**（差 vs 現実化）？ → 解釈なら interpretive/frontier。
3. その数学一致は **本物か類似か**？ → 類似なら analogy、生命語・断定語を使わない。
4. それは **imposed か emergent か**？ → 入れたものは「入れた」と書く。
5. そのスコア/次元は **飽和/副作用で偽られていないか**（非飽和・独立二重化）？
6. それは **repo で再現したか**（sandbox なら frontier）？

**Claude（サンドボックス）の監査は特にこの床を見る**：measured と frontier の混同・失敗の記録漏れ・
完成形の埋め込み・スコアの飽和・no_touch。
