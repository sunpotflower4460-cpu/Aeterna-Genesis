## 記号

**済**=measured・**再**=再現済・**部**=partial・**未**=未達（この白の天井）・**難**=困難/frontier・**—**=非該当。
括弧内 `seeded` は、その能力が**置かれた**もの（emerged でない・[`GENESIS_PROVENANCE.md`](GENESIS_PROVENANCE.md)）。

## 材料 → 能力（どの"材料"がどの能力を生むか）

| 材料（法則の性質） | 生みうる能力 |
|---|---|
| 双安定（bistable） | 局在状態・フロント・自己修復 |
| 多安定（multistable） | 複数の内部状態（→ 遺伝の素地・frontier） |
| 保存則（conserved） | 相分離・粗大化・循環 |
| 動的トポロジー | 欠陥・渦・巻き数 |
| 外部駆動（non-equilibrium） | 自発運動・循環・turnover |
| **非変分**（time-irreversible） | 持続的自発運動・自走（← SH が動かない理由の裏返し） |

## 読み方（正直に）

- **同じ "運動" が別材料から別の質で出る**：Boussinesq=coherent 対流、cgl=乱流、three_component=自走(frontier)。
  **同じ数学 ≠ 同じもの**。セルを「動く」で一括りにしない。
- **seeded は正直に併記**：SH の「単一個体」は局在が seeded。「0 から個体が創発」ではない（GPT②）。
- **空セル・未・難こそ地図の価値**：どの能力が**この白で出ないか**＝天井（[`WHITE_CEILINGS.md`](WHITE_CEILINGS.md)）。
- **「一つの白 → 普遍性クラス」**（普遍性の周期表）は深くて正しいが **frontier** とラベル。表は北極星（C4）への手段。
