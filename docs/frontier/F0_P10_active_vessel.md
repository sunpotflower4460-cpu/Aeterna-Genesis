>
> **本白は縦の創発契約（`docs/VERTICAL_EMERGENCE_CONTRACT.md`）の下流キャンペーン §5「3D Chemically
> Active Vessel」の F0 である。** P01（自走・motility）とは**別の能力**を狙う——P01 は「個体が動くか」、
> P10 は「**境界が内部反応を維持し、内部反応が境界を再生する**維持ループが、下位化学から生まれるか」。
> 縦の軸で見た標的ゲートは **I→II→III**（契約 §2）。特に **Gate III（下位実現された backreaction）**を
> Aeterna で初めて本格的に問う白——器という上位変数が内部化学という下位を変え、下位が器を保つ循環が、
> **別の外力でなく下位の局所相互作用として実現**されているかを matched control で判定する。

## 0. 中心問い
> **局所化学（複数種・化学量論・fuel/waste）と相分離から、選択的な境界が自発形成され、その境界が内部
> 反応を集中・維持し、内部反応がその境界を再生する——という維持ループが、器という上位変数を手で置かず、
> 下位の局所相互作用だけで閉じるか。** 膜を置かない・区画を置かない・内外差を置かない。

一様混合＋等方ノイズ → 相分離で液滴（器候補）→ 界面が特定化学種にだけ透過的（選択透過が界面物理から創発）
→ fuel が内部へ選択的に入り反応 → 反応生成物が界面を再生・維持 → fuel を止めると崩壊、再開で回復。
**activity を途中で on にせず、境界も透過選択性も維持ループも符号化しない。**

## 1. なぜこの白か（P01・G003 との差分と、Gate III の初挑戦）
- **G003（Model H, validated-2D candidate）**：相分離＋界面＋流れの共分化は既に role E で確立。しかし
  **単一の受動液滴**であり、化学量論・fuel/waste・選択透過・維持ループは無い（Gate I/II 止まり、Gate III 無し）。
- **P01（active droplet, F0 済）**：界面化学で液滴が**動く**（motility）。器の**維持**は問わない。
- **P10（本白）**：動きではなく**器としての自己維持**を問う。GPT 外部監査（2026-07-16）が指摘した
  「膜モデル＋代謝モデル＋複製モデルをつなぐのでなく、**同じ局所化学から内外差・境界・透過選択性・反応
  集中・エネルギー消費・成長・破損回復が同時に出る**白を探す」に対応する。
- **縦の軸での位置**：契約 §3 の橋監査で「B3（下位実現された backreaction）は一度も無い」と記録した。
  P10 はその B3 を初めて狙う——器（上位）→内部化学（下位）の下向き作用が微視物理として実現されているか。

## 2. 白（連続式・第一候補：保存化学 + Model H 相分離 + 低Re流体）

**外部レビュー（Codex, 2026-07-17）の target-encoding 指摘7件を反映した凍結版。** 各項が「選択性・反応
集中・維持ループを法則へ手で置いていないか」を実装前に閉じる（元の版は種別 D・置ける R・m が式に不在・
一様 fuel という4つの穴があった——下記で塞ぐ）。
```
# 器候補：保存相場 φ（Model H, G003 再利用・no_touch）＋化学種の φ 分配結合 χ_i
F[phi,{c_i}] = ∫[ a/4 (phi^2-1)^2 + kappa(m)/2 |grad phi|^2 + Σ_i χ_i φ c_i ] dV
mu_phi = δF/δφ = a phi(phi^2-1) - div( kappa(m) grad phi ) + Σ_i χ_i c_i
∂_t phi + u·grad phi = M_phi lap mu_phi
# 化学：fuel f, 中間体 m, waste w（化学量論 f->m->w・質量/元素保存を厳密監査）
#   ★拡散は全種で "同一の単一" D(phi)（種選択を法則に置かない・#1）。選択透過は χ_i の分配から創発。
∂_t f + u·grad f = div( D(phi) grad f ) + J_f^ext - k1 R(x) f
∂_t m + u·grad m = div( D(phi) grad m ) + k1 R(x) f - k2 m
∂_t w + u·grad w = div( D(phi) grad w ) + k2 m - k_out w
# ★fuel は器の外から（内部直接注入しない・#4）：J_f^ext = k_res (f_res - f)·1_{outer(x)}
#   1_{outer} は箱の外殻シェルのみ（φ・器の位置を参照しない固定幾何のリザーバ）。μ_f=const。
# ★反応は φ で "置かない"（#2）：R(x) = 独立に凍結した触媒場 g の局所値（例 g は φ とは別の保存量・
#   一様 IC から共分化）。第一候補では R=const（非局在）を基準にし、界面局在は emerged として測る。
# ★維持ループの下向き経路を式に陽に入れる（#3）：kappa(m)=kappa_0(1 - alpha m) と界面張力
#   gamma(m)=gamma_0 - Gamma_m m の発散形 Marangoni 応力 sigma_M[phi,m]。m が界面物理に現れる。
0 = -grad p + eta lap u + div( sigma_K[phi] + sigma_M[phi,m] ) ; div u = 0
```
- **選択透過を "置かない"（#1）**：拡散は全種**同一の単一** `D(phi)`。種ごとの透過差は、自由エネルギーの
  φ 分配項 `χ_i φ c_i` から**創発**させる。`χ_i` は**独立な物理仮定で凍結**（例：分子体積/疎水性に比例）し、
  望む選択性へ後付けで合わせない。**`χ_i=const`（非選択）を対照**に含め、選択透過が χ の差から生じたことを示す。
- **反応を "置かない"（#2）**：`R(x)` は φ で界面へ手置きしない。独立に凍結した触媒/反応物場から決める
  （第一候補 `R=const`）。界面での反応集中は**測る対象（emerged）**であって前提にしない。
  **`R=const`（非局在）対照**を含める。
- **維持ループが no-op でない（#3）**：`m` を `kappa(m)`・`gamma(m)` として界面物理へ陽に結合。これが
  「器→内部反応→器」の下向き経路の担い手であり、§4 の Gate III 対照はこの経路を操作する。
- **fuel は器の外から（#4）**：`J_f^ext` は箱外殻シェルのみのリザーバ交換。器内部へ直接注入しないので、
  内部反応が回るには fuel が選択境界を越える必要がある——`D(phi)=const` 対照で透過の寄与を切り分けられる。
- **FFT の honesty（契約・C）**：Stokes 射影に FFT を使う版は「global spectral validator」と明記し
  「中央計算なし」とは呼ばない。将来 local stencil で独立検証（P01 と同じ規律）。

## 4. 測定器の必要精密化（新規観測器・no_touch・measures.py 非改変）
- **選択透過の定量**：界面を横切る各化学種フラックス `⟨D(phi) ∂_n f⟩` を種別・内外別に測定。
  「内外で透過率が違う」を数値で（人が「膜」とラベルせず）。
- **維持ループの因果**：反応集中度（界面近傍の `k1 R(phi) f` 積分）と界面安定度（界面幅・|∇φ| の分散）の
  相互相関・時間遅れ。上位変数 U=「界面の存在/位置」を φ から**導出**（Gate I）。
- **`thermodynamic_ledger`（契約 §6・定義を凍結・Codex #6）**：フィールド名だけでなく計算を前登録する
  （さもないと動力学を変えずに収支を変えられ非再現）：
  - `matter_in = ∮_outer k_res(f_res − f) dS`（外殻リザーバからの fuel 流入・`f_res`=const 前登録）。
  - `chemical_free_energy_change`：基準状態を各種の希薄理想 `μ_i = μ_i^0 + RT ln c_i`（`μ_i^0`・`RT` を前登録）
    とし、`ΔG_rxn = Σ_i ν_i μ_i`（f→m→w の化学量論 `ν_i`）。単位は `k_B T / 格子体積`。
  - `waste_out = ∮ k_out w dV`、`entropy_production = Σ_rxn (rate·A/T) + 粘性散逸 ∫ 2η e:e dV ≥ 0`
    （A=反応親和度）。`useful_work`＝界面維持へ回った自由エネルギー（`sigma_M:∇u` の界面寄与）。
  - **測る問い**：入力（`matter_in`）を止めると崩壊、再開で回復、`entropy_production>0` が維持中持続、
    内部が入力を境界維持（`useful_work>0`）へ変換しているか。
- **Gate III matched control（契約 §2 III・向きを主張に一致させる・Codex #5）**：B3 の主張は **`器 U → 内部化学`
  の下向き**（φ の界面が化学の進み方を変える）。その下向き経路は `D(phi)`・`χ_i φ`・`R` を通じた φ の変調
  である。**主対照は「下向き経路の除去」**：`D`・`χ_i φ`・（`R` が φ 依存なら）その項を、器の**時間平均した
  空間プロファイルに固定**して「φ の揺らぎ・移動を通じた下向き変調」を消し、**化学の周辺分布・素の反応則・
  拡散強度は不変**に保つ。この下向き対照下で維持ループ（崩壊後回復）が**消え**、通常 run では**残る**ときに
  限り、`U` は下位化学**を通じてのみ**因果を持つ＝B3。**別に**、`m→界面`（上向き）フィードバックを固定する
  対照も走らせるが、これは「維持ループに反応生成物フィードバックが必要か」を示すだけで**B3（下向き実現）の
  証明にはならない**ことを明記する。無差別スクランブルは使わない（U 経由の切り分けにならない）。

## 5. 事前登録ゲート（決定的閾値・契約 §2 と整合・結果を見てから緩めない）
- **器の存在（Gate I 前提）**：連結成分=1・体積 CV<2%・非 domain-filling・界面 4–6 cell・φ から U 導出。
- **選択透過の創発（Gate I）**：内外の実効透過率比 ≥2（種別）が φ のみから生じ、IC に透過ラベル無し。
- **維持ループの有効法則（Gate II・4基準・変数も前登録・Codex #7）**：契約 §2 のデフォルト閾値を継承する
  だけでなく、この白の観測量・窓・スケールを実装前に凍結する（さもないと結果を見てから選べる）：
  - 上位変数 **`U` = φ から導出した器の状態ベクトル**＝`(体積, 重心, 界面幅, 有効半径 R)`（`U=f(φ)`・手ラベルなし）。
  - 有効法則の対象 = **`U` と界面近傍の平均反応率 `⟨k1 R f⟩_界面` の閉じた発展**（維持ループの縮約）。
  - 予測ホライズン **`Δ = τ_int`**（界面拡散時間 `≡ ξ²/M_φ κ_0`・run 開始時に確定）、評価窓 = fuel 供給定常後の
    `[2τ_int, 12τ_int]`（過渡を除外・前登録）。閾値 `R_pred<0.10`／独立 seed ≥5・係数 CV<0.15／閉性残差<0.20。
  - 粗視化 **3 スケール = ブロック線長 {2, 4, 8} セル**（線長比 4×）で有効法則の関数形不変。
  これらを Room/実験 metadata の `vertical_bridge` ブロック（契約 §6）に**実行前に**書き込み、変更履歴を残す。
- **下向き実現（Gate III）**：fuel 停止で `thermodynamic_ledger` の自由エネルギー流入が断たれ器が崩壊、
  再開で回復。かつ §4 の matched control 下で回復が消える・通常 run で残る（`U` は下位を通じてのみ因果）。
- **収束/artifact 拒否**：48³→64³→96³（fixed-L）と fixed-dx 有限箱を分離・`ε/R→0` で維持ループ onset 収束。
  diffuse-interface Marangoni/透過 artifact が界面幅 ε と共移動し `ε/R→0` で消えるなら `N: interface artifact`。
