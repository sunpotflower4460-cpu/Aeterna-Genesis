# 現在地と方向性 — Type A–D 台帳

> **確信度の勾配**：A（はっきりした）→ B（ほぼ確か・床つき）→ C（有望な解釈）→ D（価値ある未踏の仮説）。
> 各主張に tier（measured / observed / interpretive / analogy / frontier）を付け、床（ただし書き）を必ず添える。
> **創発＝物理**（本物の法則＋最小の条件→現象がひとりでに現れる）vs **入れる＝計算**（狙った答えを出す生成器）。
> 「同じ数学」≠「同じもの」。生命・脳・電子を「作った」とは言わない。
> live で触れる版は [`room/中心実験室.html`](../room/中心実験室.html)。

## 現在地 — いまどこまで創発したか

素直なルート（measured の足場でつながっている所）：

```
0≠無（差のない地面は不安定）
└ 変化が在る ＝ もう因果・時間順序が在る
  └ 低エントロピーの始まり → 時間の矢 ＋ 複雑さの窓
    └ 因果＋関係＋第三 → 整数次元の空間（d=2,3,4 を復元、d_s≈3）
      └ 渦＝物質／第三が粒子を安定化（L*∝√c4・Q_H=2・完全PDE 自己安定化）
        └ 化学・選択（turnover が淘汰を生む）
          └ 循環が内部を養う（壁つき実セル Ra_c=1714/657＝教科書<0.4%）
            └ 器の閉じ：開（駆動）かつ閉（自己維持）／空間膜小胞
              └ 循環が粒子を運び、第三が同一性を U_c まで保つ
```

**現在地**：器・循環・粒子が一つの描像に結びついた。残る本丸＝真の大域化・三体フル結合・
作用＝エントロピー・法則を 0≠無 から導く。

## 方向性 — 次に詰める（優先順）

1. **真の大域化**（Hopf）：arrested-Newton／高解像度で任意初期→L*、大 c4 まで L*∝√c4（[H001](working_ledger/H001_hopf_global_newton.md)）。
2. **三体フル結合**：代謝＋自己組織した流れ＋粒子を一つの場に、双方向結合（[H003](working_ledger/H003_three_body_coupling.md)）。
3. **作用＝エントロピー**から（Jacobson/Verlinde）：作用＝第三の役割が熱力学／もつれから出るか（[H004](working_ledger/H004_action_entropy.md)）。
4. **法則を 0≠無 から導く**（最深）：いまは法則を入れて忠実に創発させている。導出が究極。

## 🔵 Type C — 有望な解釈・構造的仮説（複数の measured を貫く）

事実そのものでなく、事実をつないで見えてくる考え方。矛盾しないが証明ではない。

<details><summary>かんたんな説明 ／ 要するに</summary>

**用語を知らなくても分かる説明**：実験で出た事実そのものではなく、事実をつないで見えてくる物語。
例えば「器は開いているからこそ閉じて自分を保てる」「変化のルールが在る＝もう時間の順番が在る」。

**要するに**：事実をつなぐ骨組み。measured と矛盾せず説明力が高いが、解釈であって証明ではない。
地図として使い、過信しない。
</details>

- **本流（throughline）**：0≠無 → 変化 → 因果/時間 → 空間 → 粒子 → 器（interpretive）。
- **第三＝最小の追加文法**：面・三和音・四次項が次元／粒子／安定化を生む（interpretive）。
- **変化のルール＝もう因果**：時間＝層状の依存、非循環性＝矢（interpretive）。
- **器＝開‐閉の二重性**：散逸的に開き組織的に閉じる（interpretive）。
- **生命の器＝宇宙、同じ経路**：違いは種パターンと閉じたループだけ（interpretive・analogy）。

## 🟣 Type D — 深い未踏の仮説（frontier）

まだ誰も解いていない最奥の問い。当たれば大きく、外れても情報。

<details><summary>かんたんな説明 ／ 要するに</summary>

**用語を知らなくても分かる説明**：まだ誰も解いていない、いちばん奥の問い。例えば「物理法則そのものが、
差の無い地面は不安定だ、という一点から導けるか？」。当たるかは分からないが、ここに進むことが一番大事。

**要するに**：最深のフロンティア。示せていないが追う価値のある問い。間違いも未科学も恐れず進む。
</details>

- **法則を 0≠無 から導く**（最深。いまは法則を入れている）— frontier。
- **作用＝エントロピーから**（第三の役割が熱力学/もつれから出るか、[H004](working_ledger/H004_action_entropy.md)）— frontier。
- **真の大域 basin**（固定格子の有界 basin を超える、[H001](working_ledger/H001_hopf_global_newton.md)）— frontier。
- **三体フル結合／開かれた進化**（代謝＋流れ＋粒子／突然変異→新規性、[H003](working_ledger/H003_three_body_coupling.md)）— frontier。
- **内生的需要でオープンエンド性**（宿主寄生者の敵対的共進化が需要を内側から上げ複雑性の頭打ちを外す、[H017](working_ledger/H017_openended_frontier.md)）— **frontier（機構を GREEN で実証）／e029**。軍拡競争は escalate/collapse で変動＝ensemble 中央値でゲート。真の無限オープンエンド性は未解決。
- **完全な大転移＝群れが再生産する高次個体**（選択の単位を群れに移す＋創設者ボトルネックで協力が高次個体に、[H017](working_ledger/H017_openended_frontier.md)）— **frontier（機構を GREEN で実証）／e029**。協力の安定化は必要条件、分業・生殖体細胞分化は未実装。
- **分業＝凸リターンが生殖体細胞分化を駆動**（群れ適応度が投資配分に凸なら専門化が創発、Michod、[H018](working_ledger/H018_frontier_solid.md)）— **frontier（機構を GREEN で実証）／e030**。専門家割合 0.26→0.85 単調、生殖/体細胞 proxy が共存(1.0)。抽象 toy 適応度＝床。
- **離散作用の平滑化で揺らぎを抑える**（BD 生作用は N とともに揺らぎ支配＝H008、Glaser-Surya 平滑化が 1/7〜1/40 に damp、[H018](working_ledger/H018_frontier_solid.md)）— **frontier（障害を GREEN で定量化）／e031**。曲率信号は ~1σ＝床。**治療でなく障害の定量化**。
- **3D（2+1）DS の d>2 病理**（面積則は成立(R²>0.85)だが係数が密度でドリフト(4.02×)＝2D の純数と違い普遍性を失う、[H018](working_ledger/H018_frontier_solid.md)）— **frontier（病理を GREEN で定量化）／e032**。なぜ 2D が SOLID・3D が床か。Barton cured 分子は未再現＝床。
