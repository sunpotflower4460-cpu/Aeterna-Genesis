# H019 — エージェントを場へ（規則に従う個体を、本物の場の方程式の創発に置き換える）

- **状態**: **進行中（e033 GREEN・Type A 忠実）**。目標＝リポジトリ内の「エージェントモデル」を
  一つずつ**場の物理**に作り直し、同じ定性事実を規則でなく物理から出す。
- **起票者**: Claude Code / 2026-07-10
- **狙う Type**: A（忠実な創発）。本物の場の方程式を入れ、最小条件から現象が勝手に出る。
- **動機**: LAW.md §0 は (A) 忠実な創発を積む。だが一部の実験は**エージェントモデル**
  （個体＋選択・再生産・戦略コピー）で、これは場でない＝**床**（e027 openended.py 自身が
  `THE TRAP: an AGENT model, NOT a field -- a Floor` と告白）。エージェントの規則を、
  **自由エネルギーの勾配流・反応拡散**などの本物の連続場の法則に置き換えられるか。

## e033（分業＝スピノーダル分解）— GREEN
- **入れたもの**: Cahn-Hilliard 保存勾配流 ∂φ/∂t=M∇²(f'(φ)−κ∇²φ)、Flory-Huggins
  f(φ)=φlnφ+(1−φ)ln(1−φ)+χφ(1−φ)、一様 0.5＋微小ノイズ。ノブは相互作用 χ のみ。
- **出たもの**: 臨界点 **χ_c=2**（Flory-Huggins 値・入れていない）で相分離、χ>2 で二相共存
  （体細胞 proxy φ_low ＋ 生殖系列 proxy φ_high）、共存組成が**理論 binodal と定量一致**
  （0.15/0.85, 0.07/0.93）、**ドメイン波長**が創発。エージェント・選択・再生産なし。
- **verdict**: 分業（専門化＋共存）は**場ネイティブに実現できる**。e030 のエージェントは一つの基層、
  Cahn-Hilliard 場はもう一つ。**同じ数学≠同じもの**なので「進化＝相分離」とは言わない——
  同じ定性事実が規則なしの物理から出る、という主張に限る。

## メモ
「エージェントを場に翻訳した」でなく「同じ創発事実（凸性→専門化＋共存）が本物の場の方程式から
規則なしで出る」を数で（χ_c=2・binodal 一致・波長）。誇張なく、床つきで。作為なく、検証しながら、一つずつ。
