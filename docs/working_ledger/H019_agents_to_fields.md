# H019 — エージェントを場へ（規則に従う個体を、本物の場の方程式の創発に置き換える）

- **状態**: **進行中（e033 GREEN・Type A 忠実）**。目標＝リポジトリ内の「エージェントモデル」を
  一つずつ**場の物理**に作り直し、同じ定性事実を規則でなく物理から出す。
- **起票者**: Claude Code / 2026-07-10
- **狙う Type**: A（忠実な創発）。本物の場の方程式を入れ、最小条件から現象が勝手に出る。
- **動機**: LAW.md §0 は (A) 忠実な創発を積む。だが一部の実験は**エージェントモデル**
  （個体＋選択・再生産・戦略コピー）で、これは場でない＝**床**（e027 openended.py 自身が
  `THE TRAP: an AGENT model, NOT a field -- a Floor` と告白）。エージェントの規則を、
  **自由エネルギーの勾配流・反応拡散**などの本物の連続場の法則に置き換えられるか。

## e033（分業＝スピノーダル分解）— GREEN
- **入れたもの**: Cahn-Hilliard 保存勾配流 ∂φ/∂t=M∇²(f'(φ)−κ∇²φ)、Flory-Huggins
  f(φ)=φlnφ+(1−φ)ln(1−φ)+χφ(1−φ)、一様 0.5＋微小ノイズ。ノブは相互作用 χ のみ。
- **出たもの**: 臨界点 **χ_c=2**（Flory-Huggins 値・入れていない）で相分離、χ>2 で二相共存
  （体細胞 proxy φ_low ＋ 生殖系列 proxy φ_high）、共存組成が**理論 binodal と定量一致**
  （0.15/0.85, 0.07/0.93）、**ドメイン波長**が創発。エージェント・選択・再生産なし。
- **verdict**: 分業（専門化＋共存）は**場ネイティブに実現できる**。e030 のエージェントは一つの基層、
  Cahn-Hilliard 場はもう一つ。**同じ数学≠同じもの**なので「進化＝相分離」とは言わない——
  同じ定性事実が規則なしの物理から出る、という主張に限る。

## e034（空間協力＝双安定反応拡散フロント）— GREEN
- **入れたもの**: 双安定 Nagumo/Schlögl 反応拡散 u_t=D∇²u+u(1−u)(u−a)、協力密度 u∈[0,1]。ノブは a のみ。
  平均場（混合）＝同じ反応の∇²除去版。
- **出たもの**: 平均場は閾値下シードで崩壊（u→0）、空間は協力ドメインが a<1/2 でフロント侵入（生存）、
  a>1/2 で裏切り侵入。**創発閾値 a_c=1/2（Maxwell 点・入れていない）**で符号反転、速度が **Nagumo 則
  √(D/2)(1−2a) と定量一致**（誤差<0.01）。エージェント/模倣/利得計算なし。
- **verdict**: e027 の三事実（混合崩壊/空間生存/条件閾値）が場から出る。**同じ数学≠同じもの**——
  協力の進化＝双安定フロントとは言わない。協力の**安定性**であって完全な大転移でない（床）。

## e035（Red Queen＝捕食者被食者 Hopf 分岐）— GREEN
- **入れたもの**: Rosenzweig-MacArthur 捕食者被食者場（Holling II）、連続密度 H,P、ノブは富栄養化 K のみ。
- **出たもの**: K<K_c で頭打ち（固定点安定＝摂動減衰）、K>K_c で内生持続振動（リミットサイクル＝
  外部強制なしの Red Queen）。**発生点が解析 Hopf 点（Jacobian trace=0）と一致**（誤差 0.006）、位相ずれ振動。
  m・H0 で Hopf 点が動いても発生点が追従（0.7→0.62→0.82）。エージェント/ゲノム/選択サンプリングなし。
- **verdict**: e029 の「静的は頭打ち／相互作用は内生持続変化」が場から出る。**床**：場が与えるのは内生
  **振動**であって、文字通りの無限複雑性上昇ではない（それは依然 frontier）。

## e036（適応＝レプリケーター突然変異場）— GREEN
- **入れたもの**: replicator-mutator（Crow-Kimura）∂p/∂t=D∂²p/∂x²+(f−⟨f⟩)p、形質軸上の密度場、
  ノブは突然変異 D と適応度地形 f のみ。移動最適点は共動座標系で扱う。
- **出たもの**: 平衡分散が閉形式 **σ²=√(2D/k)** と一致、ピーク外初期分布がピークへ登る（適応）、
  移動最適点の追跡遅れが **L=v/(kσ²)**（lag load）で速度に線形。すべて閉形式と一致（誤差<0.003）。
  エージェント/サンプリングなし。k・D で分散・遅れが理論曲線上（頑健）。
- **verdict**: e027 の適応・突然変異選択平衡・追跡ラグが場から出る。**床**：主張は小速度の線形 lag 領域のみ
  （臨界速度超で個体群喪失）。**同じ数学≠同じもの**——進化＝RM 方程式とは言わない。

## 保留（正直に・場化しなかったもの）
- **e027 openended（可変ゲノム複雑性上昇）**：可変長ゲノム＝離散構造で、忠実な連続場化は非自明。
  無限オープンエンド性は H017/H018 で frontier のまま。**誇張して場化したと言わない**。
- **e029 major_transition（群選択＋ボトルネック）**：多層選択＝離散的な群構造＋創設者ボトルネックで、
  純粋な場としての忠実版は非自明（e034 が「空間構造で協力生存」は場化済み）。frontier として保留。
- **e004 octave（自己申告 analogy）**：MERA/AdS テンソルネットワークの忠実な物理版は研究 frontier。
  クリーンなゲートに落ちないため保留。**「物理にした」と偽らない**。

## メモ
「エージェントを場に翻訳した」でなく「同じ創発事実が本物の場の方程式から規則なしで出る」を数で
（e033: χ_c=2・binodal 一致・波長／e034: Maxwell 点 a=1/2・Nagumo 速度／e035: Hopf 点・持続振動）。
誇張なく、床つきで。作為なく、検証しながら、一つずつ。場化できないものは正直に保留する。
