# H003 — 三体フル結合（代謝＋自己組織した流れ＋粒子を一場に双方向結合）

- **状態**: resolved (**promoted**)
- **起票者**: Claude Code / 2026-07-01
- **狙う Type**: B/C（床つき measured の結合＋解釈）
- **依拠する既知事実**: e016（粒子＝ Q_H 保持の有限 L* ホップ粒子）、e017/e013（自己組織した
  循環が閾値で立ち内部を養う）、e018（駆動される器＝開‐閉の二重性、切ると死）、
  e019 Stage1（規定 roll が粒子を運び、第三が U_c まで同一性を保つ）。

## 仮説
三者を**双方向**に結ぶと、一方向では出ない創発が出る：**代謝が流れを駆動 → 流れが粒子を運ぶ →
粒子が流れに背反応（抗力）** の閉ループ。とくに粒子の背反応が流れを**自己制限**し、
粒子が引き裂かれない準定常（ホメオスタシス）に落ち着くか。駆動を切ると流れが止み、
器ごと死ぬ（三体が一蓮托生）。

## 試験（方法）
`experiments/e019_coupling/three_body.py`。最小三体：代謝 b・流れ振幅 U・実 3D ホップ粒子 n。
`db/dt=s(1-b)-kU·U·b`（駆動 s が代謝供給、流れが消費）、`dU/dt=gU·b-dU·U-drag·P(n)`
（代謝が流れを駆動、粒子コヒーレンス P が抗力＝背反応）、n は振幅 U(t) の roll で移流＋安定化流。
測る：駆動あり→(b,U) が準定常で粒子保持（Q_H≈1）＝自己制限か；駆動オフ→U→0 で輸送停止（死）。
背反応 drag=0（一方向）との対比で「双方向でしか出ない」を確認。

## 結果（`three_body.py` → `results/three_body.json`）
- **双方向（drag>0）**：背反応が流れを**自己制限**（U=3.35）し粒子保持（Q_H=0.97）。
- **一方向（drag=0）**：背反応なし→同じ代謝が U=9.56 まで過駆動→**粒子破壊（Q_H=−0.21）**。
  ＝**背反応が粒子を救う**創発的な負のフィードバック（backreaction_saves_particle=True）。
- **駆動オフ（s=0）**：b→0→U→0→流れ停止・輸送停止＝三体が一蓮托生。
- `not_scripted_check`：Q_H は B=curl A、U/b は ODE、自己制限は入れていない。
- 床：流れは振幅還元（完全 NS でない）、roll 形は規定、「ホメオスタシス/一蓮托生」は interpretive。
