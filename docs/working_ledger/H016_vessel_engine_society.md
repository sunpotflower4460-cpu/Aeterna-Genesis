## e024 の結果（GREEN・物理量ゲートのみ）
**Stage1 ratchet**（`ratchet.py`）：非対称周期ポテンシャルが対称ノイズを整流。
整流 (v(+A)+v(−A))/2 @asym=1,A=4.5 = **+0.60**（対称 ~0）、A 1.5→6 で **0.02→0.96**（単調増）、
asym=0.1（51/49）で **+0.08**（閾値なし）。罠＝静的傾きを別々に測って平均。
**Stage2 motor**（`motor.py`）：等間隔 N 位相の場は N≥3 で回転（ripple<0.05）・N≤2 は脈動；
三相電流和 max|Σ|=0；トリプレン零相＝[3,6,9]；三相ロータ自己回転 **+19rev**（単相はぐらつき、単相+ラチェットで回る）。
**Stage3 vessel_motor**（`vessel_motor.py`）：燃料勾配 Δμ が三回対称ロータを回し（閾値→上昇）、
負荷に停動（load≈3）、器 **Ψ=0.93** を維持、燃料切断で **Ψ→0**（死）、方向は燃料の符号で反転（±0.56）。
罠＝産物は**平滑化正味回転** `om_s` ＋ `max(om_s−0.05,0)`（`max(dφ/dt,0)` はノイズ床を拾い死なない）。
