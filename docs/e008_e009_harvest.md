# e008 / e009 ハーベスト — 白から何が、作為なく、創発したか

一つの白（一様＋ノイズ）と忠実な局所法則だけから、**物質・時間・空間・器・複製**が
どこまで入れずに出るかを、回せるだけ回した収穫。`measured` / `interpretive` /
`analogy` / `frontier-observation` を厳密に分け、**全ての床を明記**する。

## 1. 創発したもの（tier つき）

### measured（起きた事実・数で）
- **KZ 渦が白から自発核生成**：N ∝ τ_Q^{−b}、遅クエンチ域 **b≈0.46**(R²=0.92)。正味巻き≈0（±対）。循環量子化。
- **速クエンチ飽和（KZ の膝）**：狙っていない KZ 特徴が出た（監査4の本領）。
- **粗大化の矢**：クエンチ後 N が減り欠陥間スペーシングが 8→28 へ単調に伸びる。
- **可逆 echo**：保存 GPE で戻り忠実度 **1.0000**。→ 矢は法則に無い。
- **3D 量子化渦線**：白から創発、線数は τ_Q で減少（3D KZ）。**収縮可能率 ≈0.30**（残りは**トーラスを巻く**閉じた線）。旧「閉率0.94」は非周期ラベリングのアーティファクト（§3で訂正）。
- **持続トーラス電流**：量子化循環が持続、ノイズでも位相すべりせず保護（E drift~1e-13）。
- **種→自己複製**：Gray-Scott mitosis で 1種→**16成分**。**枝分かれ**：フラクタル次元 **1.55**。**勾配→方向性成長**：重心が −8.6 セル ドリフト。
- **基板普遍性**：相対論的複素スカラー（非GPE）でも KZ 渦、**b≈0.50**。
- **欠陥の化学（±対形成）**：逆符号渦が同符号より近い（**8.6 vs 13.5**、周期境界=最小像距離で測定）。**符号入替えの帰無仮説に対し有意（p<0.003）**＝引力で対を作る。

### interpretive（意味の読み）
- 物質・時間・空間の「**同時共創発**」（measured は各創発と KZ 数）。
- 矢は**散逸（粗視化）＋低S始端**から（保存法則では echo が戻る）。
- 普遍性 → うえきの「共通なら硬い」が一歩硬くなる。

### analogy
- 持続電流＝「打ち消さない循環」は **electron-like**。GS 形態は **plant-like**。いずれも記述であって電子・生命ではない。

### frontier-observation（予測外・未解明だが観測）
- **基板普遍性**（GPE と相対論的スカラーが同じ質の KZ 渦）— 予想以上にクリーン（NLKG は ρ≈1 に綺麗に凝縮）。
- **±渦の対形成**（"分子"の芽）— **置換検定で有意**（p<0.003、幾何的偶然でない）。
- **Hopf 的リンク渦輪は縮む**（素の GPE に安定化なし）。

## 2. 驚き（surprise log）の上位

1. **基板普遍性**：別の法則（相対論的 Klein-Gordon）でも白から同じ KZ 渦が出た。共創発が GPE 固有でないことの強い証拠。指数は z で変わる（GPE z=2→b≈0.46、相対論 z=1→b≈0.50）が、**質（負べき＋渦＋正味ゼロ）は不変**。
2. **±渦の対形成**：逆符号が同符号より明確に近い。狙っていない"化学"（束縛）の芽。
3. **速クエンチ飽和**：KZ の膝を狙わず観測。単一べきで無理に通さず、遅域フィット＋飽和フラグで正直に分けた。

## 3. 負の結果・崩壊・縮小（隠さない＝手がかり）

- **Hopf的リンク渦輪は縮む**（2368→931）。素の GPE に安定化（高階微分＝「第三」）が無いから。→ 持続させるには安定化項が要る、という**何が足りないか**の手がかり。
- **3D は部分凝縮**（太い芯）。閉/貫通の分類はボクセル位相プロキシで、厳密な渦線トレースでない。より長い hold／細格子が要（スケール候補）。
- **自分のバグを1つ捕獲**：減衰 γ=0.1 は場が**未凝縮**（ρ≈0.02）で、欠陥カウントが**音波**を拾っていた。ρ≈μ/g を確認してから（γ=0.3）測り直した。数が物理と合わない時、まずコードを疑う、を実践。
- **レビューでもう1つ捕獲（周期性アーティファクト）**：3D の「閉率 0.94」は **非周期** `ndimage.label` がトーラスを巻く渦線を境界で分断し、小ループに誤計数した**水増し**だった（Codex P2）。周期 union-find で測り直すと**収縮可能率 0.30**＝大半は巻き線。主張を「量子化渦線が創発＋3D KZ」に限定し、「ほぼ閉ループ」は撤回。**境界条件と測定法の不一致が偽の GREEN を作る**実例。同じレビューで、(a) Hopf の縮小判定が一様初期密度（baseline=0）で**符号が逆**になっていたのを修正（縮む=True に）、(b) ±対形成の距離を**最小像（周期）**に修正（結論は不変：有意 p<0.003）。

## 5. 床（全て明記・必須）

1. **固定格子＝空間は与えられている**。ここの「幾何」は相関/折りたたみであって、もつれ(Ryu-Takayanagi)幾何ではない。量子ルート(e005-e007)が相補的な片割れ、橋渡しは **frontier**。
2. 「**同時共創発**」は interpretive（measured は各創発と KZ 数）。
3. **KZ 指数 b は protocol 依存**（γ・閾値・遅域フィット・z）。普遍指数は主張しない（普遍なのは「負べき＋渦」という質）。
4. **「電子的/植物的」は analogy**。電子・生命を作ったのではない（禁止語）。
5. Tier C（Hopf・±分子・普遍指数）は **frontier-observation**：縮む・スナップショット・z依存は失敗でなく情報。
6. 背景なし・もつれ幾何での真の同時共創発・**器の閉じ**は最前線（未達）。

## 6. 最も興味深いもの（上位）＋ 次に試すべきこと（優先順）

**見つかった最も興味深いもの**
1. 基板普遍性（共創発が法則の細部に依らない）。
2. ±渦の対形成（欠陥の"化学"の芽）。
3. 持続トーラス電流のトポロジー保護（打ち消さない循環）。

**次に試すべきこと（優先順）**
1. **±分子の束縛を動的に**確かめる（対の寿命・有限温度での解離＝「欠陥の化学」を measured に）。
2. **Hopf 安定化**：Faddeev-Skyrme 型の高階微分（「第三」）を入れて Hopfion が持続するか（縮む→持続の境界）。
3. **持続電流 × 成長の結合**（器＋中身）：トーラス循環を養分輸送路にして種が乗って育つか（open-5、要スケール）。
4. **普遍性の指数を z で説明**：GPE(z=2) と相対論(z=1) の b の差が KZ 予測 b=(D−d)ν/(1+νz) と合うかを定量照合（measured へ）。
5. **3D 渦線の厳密トレース**（細格子・長 hold）でループ/リンク数を上げ、トーラスの位相を精密化。
6. **駆動系（CGL）**で持続する非平衡相（閉じる/開くのバランス、器の閉じへ）。
