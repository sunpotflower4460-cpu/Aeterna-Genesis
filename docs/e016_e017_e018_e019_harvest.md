## 1. 創発したもの（tier つき）

### measured（起きた事実・数で）
- **Hopf size 則**：安定化流の収束 size が √c4 則（size=k√c4、L=44/κ=40 較正で k=0.901・CV=3.5%・R²=0.941、
  k は c4 で ~8% 緩く低下＝軽い sub-√c4）、各 c4 で Q_H≈1。**Q_H=2**（方位角巻き×2）も同じ流れで保持（|Q_H|≈2）、エネルギー単調。
- **壁つき臨界 Rayleigh 数**：no-slip 1713.9@a_c3.12／free-slip 657.3@2.22＝教科書（1707.76/657.5）と**<0.4%**。
  周期箱 Ra_c≈20（e013）は箱固有値の artifact。
- **壁つき DNS**：Nu~1（伝導、Ra<Ra_c）→ Nu>1 上昇（対流、Ra=5000 で Nu≈2.1）、
  内部 delivery が対流で 0.017→0.055（伝導は内部を飢えさせ、対流が養う）。
- **三腕オートポイエーシス＋相図**：基質→代謝→膜の三腕、どの腕/駆動を切っても死、
  臨界駆動曲線（bracket された行）(dA,s_c)=[(0.25,0.05),(0.35,0.1)]（strictly rises、dA=0.15 は走査域外＝報告のみ）。
- **空間膜小胞**：phase-field で境界をもつ有界単一小胞（inside_frac≈0.28、薄膜 band≈0.05、連結1）が
  駆動で持続・**駆動を切ると溶ける（死）**、漏れ↑→生存駆動↑（0.1→0.2、生存判定=bounded_single_vesicle）。
- **循環×粒子（輸送＋U_c）**：規定 roll が粒子を運び centroid が U で単調移動、
  第三が Q_H を U_c≈7 まで保持、超えると引き裂かれ Q_H→0・size 発散。
- **★三体フル結合**：代謝↔流れ↔粒子の閉ループで、**粒子の背反応が自己組織流を自己制限**（U を 3.35 に抑え）
  粒子を保持（Q_H=0.97）。**背反応を切ると（一方向）流れが U=9.56 まで過駆動され粒子は破壊（Q_H=−0.21）**
  ＝背反応が粒子を救う創発的な負のフィードバック。**駆動オフで b→0.005・U→0・輸送停止＝三体が一蓮托生**。

### observed / frontier-observation（予測外・観測）
- **高解像度での fit 劣化＋検証した仮説の否定（負の結果）**：size 則は L=44 と L=52 で tight
  （CV≈3.5%）だが L=56 で sub-√c4（CV≈6.6%・R²≈0.57）。**κ∝dx⁴ 過減衰**を疑い H001（arrested-Newton）で
  検証したが、L=52 で較正 κ（CV3.7%）は固定 κ（CV3.5%）を**改善せず**＝**単純 κ スケールでは説明できない**
  （dead-end）。高解像度の劣化は**開いた床**（原因未特定）。＝仮説を数で検算し、外れを誇張せず記録。
- **Hopf basin は有界**（両側）：保持 start は有限窓。大域でない（床）。
- **三体の自己制限（ホメオスタシス）**：背反応が流れを抑え粒子が生き延びる負のフィードバックは
  一方向結合では出ない＝双方向でしか出ない創発（interpretive で「ホメオスタシス」）。

### interpretive
- 「器＝開‐閉の二重性」を空間膜（小胞）で実体化。三体は「代謝・流れ・粒子が一蓮托生」。
- 「第三」（四次項）が粒子の同一性を輸送・シアに抗して保つ（U_c まで）。

### analogy
- 循環＝血流/心臓、膜/小胞/代謝＝protocell、ホップ粒子＝「粒子」、自己制限＝ホメオスタシス。
  いずれも記述で、心臓/生命/電子/細胞ではない。

## 2. 驚き（surprise log）の上位
1. **★三体の自己制限**：粒子の背反応が自己組織流を抑え、粒子が引き裂かれずに生き延びる
   （一方向では崩壊）＝双方向結合でしか出ない負のフィードバック。
2. **膜小胞が駆動で生き、切ると溶ける**：0次元動力学でなく**空間の器**が創発。
3. **仮説を検証して否定できた**：高解像度の size 則劣化を κ∝dx⁴ 過減衰と疑ったが、検算で否定
   （較正 κ は改善せず）＝「自分のバグを疑う」を実践し、外れた仮説を誇張せず記録（開いた床）。
4. **Q_H=2 も同じ第三で保持**：高電荷粒子も同じ安定化子で生きる。

## 3. 負の結果・崩壊・床（隠さない＝手がかり）
- **e016 の L=56 は sub-√c4（CV≈6.6%・R²≈0.57）**：原因未特定の開いた床（κ∝dx⁴ 仮説は検証して否定）。L=44/52 は tight（CV≈3.5%）。真の大域化は H001。
- **Hopf basin は有界**（両側、大域でない）、大 c4 は解像度ぎりぎり。
- **e017 DNS は粗い 2D・弱超臨界**（乱流でない）、固定格子。
- **e018 は analogy**（protocell でない、連続 phase-field）。
- **e019 は規定 roll／振幅還元流**（完全 NS でない）、U_c は crossover。三体の流れは 0 次元振幅。
