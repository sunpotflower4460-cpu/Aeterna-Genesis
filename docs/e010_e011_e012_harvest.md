## 1. 創発したもの（tier つき）

### measured（起きた事実・数で）
- **KZ 間隔はコヒーレンス長で決まる**：間隔/ξ = **2.75（CV=0.041）**が 16倍の τ_Q 範囲で一定。
- **KZ 内部整合 2σ=b**：ξ∝τ_Q^0.282、N∝τ_Q^{−0.570}、2σ=0.564 ≈ b=0.570（**相対差 1%**）。全凝縮(ρ 0.94–0.99)で締め。
- **z 順序 b(z=1) > b(z=2)**：軽い減衰の自然 z プロトコル（γ と η は別演算子なので「整合」でなく各基板の自然 z を保つ）で NLKG(z=1) b≈0.52 > GPE(z=2) b≈0.24（KZ b=(D−d)ν/(1+νz) と同じ向き）。
- **渦の選択的ハミルトニアン則**：± 双極子は**並進** v·d=**0.67**(CV≈0.01)、++ 同符号は**公転** ω·d²=**2.08**≈点渦の 2(CV≈0.04)。逆符号は並進・同符号は公転＝選択的。
- **熱解離**：確率的GPE で ±対の寿命が T とともに**単調減少**（2500→1985、drop 21%、slope≈−780）。
- **Hopf 不変量の整数性**：滑らかな構成から **Q_H=1.00**（Whitehead (1/16π²)∫A·B）。
- **Derrick 安定化**：素(c4=0)は崩壊、第三(c4>0)で**有限 L*=√(c4 E4/E2)**、**L*∝√c4**（0.565:1.130:1.696 = 1:2:3）。
- **動的な素の崩壊**：勾配流で素(c4=0)は **Q_H→0（崩壊）**＝e009 の縮みの動的確認（robust）。エネルギー単調減で勾配を検算（離散EL、ΔE/予測≈1.0）。

### interpretive（意味の読み）
- KZ 機構（間隔がξで決まる）＋ z 普遍性（小 z→大 b）。絶対指数は protocol 依存。
- 「第三＝最小の追加文法（勾配→四次）」が安定化子（静的 Derrick で有限 L* が確か；動的では素の崩壊を確認、抵抗は脆く frontier）。
- 熱解離は Arrhenius 的（τ~exp(ΔE/T)、ΔE≈0.02）— 低T cap・高T BKT の床つき。

### analogy
- ±/++ 束縛は「分子/化学」**的**、ホップ粒子は「粒子」**的**、持続循環は「電子」**的**。いずれも記述であって、分子・電子・生命ではない。

### frontier-observation（予測外・未解明だが観測）
- **動的な「第三」の崩壊抵抗は脆い**：細格子（L≈36）では最終 Q_H が c4 で増える（部分抵抗）が、粗格子/陽的 dt では c4>0 も同程度に崩壊・数値不安定（Q_H が負に振れることも）。よって**抵抗は GREEN ゲートにせず frontier**（作為で通さない）。robust なのは素の崩壊とエネルギー単調減。
- **完全な3D動的自己安定化（Q_H≈1 を保つ持続）は未達**：四次項の剛性で陽的 dt が極小、格子カットオフが L* と競合。**安定化の確かな証拠は静的 Derrick の有限 L* 最小点**（汎関数の性質）。**細格子/陰的(Fourier)処理がスケール候補**。

## 2. 驚き（surprise log）の上位
1. **KZ 内部整合 2σ=b を 1% で**：間隔とξが同じ指数で結ばれ、欠陥が本当に凍結長で決まることを数で。
2. **一つの場の方程式から二つの選択的則**：逆符号=並進(v·d)、同符号=公転(ω·d²=2)。狙わず出た「化学の選択性」。
3. **Q_H が滑らかな構成から整数**：丸めていないのに Whitehead 積分が 1.00。
4. **L*∝√c4 を厳密に**：Derrick の予測どおり（c4=1:4:9 で L*=1:2:3）。

## 3. 負の結果・崩壊・縮小・confound（隠さない＝手がかり）
- **z 対比は減衰に敏感**：重い減衰は相対論的場を緩和的(z=2)にし、b の対比を消す。順序は自然 z（軽い減衰・単調域）でのみ出る。→ z の役割そのものの手がかり。
- **有限T解離の confound**：高 T は BKT で背景渦が増殖し、対の追尾が崩れ寿命が非単調に。→ clean 窓は中間 T、tight window 連続性追尾で個体を追う（設計者の罠 c を実践回避）。
- **Hopf 動的完全安定化は未達**：四次項の剛性で dt 極小・格子カットオフが L* と競合。**細格子では部分抵抗が見えるが頑健でなく（粗格子では崩壊・数値不安定）、Q_H≈1 持続は出ない**。失敗でなく「何が足りないか（解像度/陰的処理）」のデータ。
- **静的 E2,E4 は解像度(dx)依存**：スペクトル微分では UV 発散的。FD で安定値を採るが絶対値は scheme 依存（Q_H と Derrick の形は robust）。

## 5. 床（全て明記・必須）
1. **固定格子＝空間は与えられている**。KZ・渦・ソリトンの物理で、もつれ(RT)幾何ではない（量子ルート e005-e007 が相補）。
2. **KZ 絶対指数 b は protocol 依存**（γ・閾値・hold・凍結窓・z）。measured は機構（間隔∝ξ）・内部整合（2σ=b）・z 順序。
3. **「分子/化学/粒子/電子」は analogy**。束縛渦対・ホップ粒子であって、分子・電子を作ったのではない。「同じ数学」≠「同じもの」。
4. **有限T解離**は低T cap 打ち切り・高T BKT の床つき（Arrhenius は interpretive）。
5. **Hopf 動的完全自己安定化は frontier**（未達）：解像度/陽的 dt の限界。静的 Derrick で有限 L* の存在は確認済み。
6. **静的 E2,E4 は解像度(dx)依存**（Q_H・Derrick の形が robust）。
