# e008 — 同時共創発（白 → 物質・時間・空間）— AUDIT

> 一つの白（一様＋ノイズ）と一つの局所法則（GPE）だけから、物質（位相欠陥）・
> 時間（矢＋echo）・空間（相関長）が、どれも入れずに出るかを測る。
> **各創発と KZ 数は measured。「時空の同時共創発」という統合は interpretive。**
> 固定格子＝空間は与えられている（床）。

## 監査ヘッダ（run.py 先頭と同一）

```
MODULE:   e008_coemergence
QUESTION: 白(一様+ノイズ)＋一つの局所法則(GPE)だけで、物質(位相欠陥)・時間(矢+echo)・
          空間(相関長/折りたたみ)が同時に創発するか。3Dで閉じた渦線(トーラス)が出るか。
PUT IN:   減衰GPE dψ/dt=-(i+γ)(-½∇²+g|ψ|²-μ)ψ ＋ 白(小ノイズ) ＋ クエンチ(μを<0→>0,τ_Q)。
          echo は γ=0(保存GPE)。渦・矢・幾何は一切書かない。これだけ。
EMERGED:  (measured) KZ渦が白から自発核生成 N~τ_Q^{-b}、正味巻き~0(±対)、粗大化の矢
          (Nが減りスペーシングが伸びる=保存法則では出ない)、長さスケール。
CLAIM TIER: measured(欠陥数・KZ指数b・正味巻き0・量子化・粗大化・echo忠実度) ;
            interpretive(時空物質の同時共創発・矢は散逸由来の教訓) ;
            frontier(もつれ(RT)幾何・背景なし)。
STATUS:   GREEN（各 measured 創発が全監査通過。統合の読みは interpretive。床つき）
A_OR_B:   (A)忠実な創発。手入力=場の法則＋白。背景(格子=空間)は与えている((B)未着手)。
```

## 基準測定（`result.json`、L=192, γ=0.3, μ:−1→+1）

### 物質（Kibble-Zurek 渦＝「並行する渦宇宙」）
| τ_Q | N_def（3 seed 平均） | 正味巻き |ave| |
|---|---|---|
| 50 | 304.7 ±34.5 | 7.3 |
| 100 | 266.3 ±11.1 | 8.3 |
| 200 | 243.0 ±17.6 | 13.7 |
| 400 | 220.3 ±9.6 | 3.0 |
| 800 | 157.3 ±11.0 | 2.0 |
| 1600 | 93.7 ±10.1 | 1.7 |

- **KZ べき則（遅クエンチ域）**：N ∝ τ_Q^{−**0.461**}（log-log R²=0.920）— measured。
- **速クエンチ飽和（KZ の膝）**：True（最速 τ_Q で N が power-law 外挿より下＝有限の最大欠陥密度）。狙っていない KZ 特徴が出た（監査4）。full-range b=0.312。
- **正味巻き ≈ 0**（±対で生成＝陰陽の保存対）— measured。**循環量子化**＝True。

### 時間（矢＋echo＋内部時計）
- **粗大化の矢**：クエンチ後、N が単調に減り、欠陥間スペーシング L/√N が単調に伸びる（8.2→28.0）— measured。
- **可逆 echo**：保存 GPE（γ=0）で前進→共役→前進の**戻り忠実度 1.0000** — measured。
  → **矢は可逆な法則の中に無い**。粗大化（散逸＝粗視化）＋低S始端から出る（interpretive な教訓、e006/Phase1 と同じ）。

### 空間（相関/折りたたみ幾何）
- 欠陥間の**創発的な長さスケール**（スペーシング／相関長）が時間とともに伸びる。
- 折りたたみ（オクターブ階層、e004）との接続は構造として（interpretive）。
- **床**：これは相関/折りたたみ幾何で、もつれ(RT)幾何ではない（固定格子）。

### 3D（トーラス＝閉じた渦線、`run3d.py` → `result3d.json`）
| τ_Q | 閉ループ | 貫通線 | 閉率 |
|---|---|---|---|
| 100 | 19 | 1 | 0.95 |
| 200 | 17 | 1 | 0.94 |
| 400 | 11 | 1 | 0.92 |

- **閉じた渦ループ（トーラス）が白から創発**、平均閉率 **0.94**、ループ数は τ_Q で減少（3D KZ）— measured。
- 床：部分凝縮（太い芯）のためボクセル位相のプロキシ。「トーラス」は閉ループという位相の事実で、時空の主張ではない。

## 7監査
| # | 監査 | 判定 | 理由 |

| 1 | 規則が結果を名指す? | **No** | GPE は局所、渦/向き/幾何を書いていない |

| 3 | 結果は初期条件に? | **No** | 白(一様＋ノイズ)。渦・矢・幾何は後から |

| 6 | 頑健? | **Yes** | L(128/192/256)・γ(0.2/0.3/0.5)・seed・次元(2D/3D)で power law 持続（`robustness.json`） |

→ 各 measured 創発は全監査通過＝**GREEN**。ただし下記の床。

## 床（隠さない・必須）
1. **固定格子＝空間は与えられている**。ここの「幾何」は相関/折りたたみであって、もつれ(Ryu-Takayanagi)幾何ではない。量子ルート(e005-e007)が相補的な片割れで、橋渡しは frontier。
2. 「物質・時間・空間の**同時共創発**」は **interpretive** の読み。measured は各創発と KZ 数に限る。
3. **KZ 指数 b は protocol 依存**（γ・カウント閾値・遅域フィット）。遅クエンチ域で b≈0.46、速域は飽和。普遍指数の主張はしない（measured は「負べき＋飽和」という KZ の形）。
4. **矢は散逸(γ>0=粗視化)＋低S始端**から。保存法則(γ=0)では echo が戻る（矢は法則に無い）。
5. **3D は部分凝縮**（太い芯）。閉/貫通の分類はボクセル位相プロキシで、厳密な渦線トレースではない。
6. 背景なし・もつれ幾何での真の同時共創発・器の閉じは **frontier**（この実験では未達）。
