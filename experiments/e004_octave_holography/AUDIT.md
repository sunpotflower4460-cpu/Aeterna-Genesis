## 測定結果（`result.json`、L0=512）

### Part A — 折り畳み階層は双曲か
| 量 | 測定値 | 解釈 |
|---|---|---|
| ボール成長 R²（指数 vs べき） | **0.9825 vs 0.9616** | 指数が勝つ＝双曲（控えめ差） |
| ボール成長の形 | 1,4,9,16,26,40,60,88,128,… | 局所は ~r²（2D的）→大域で超多項式＝双曲 |
| 直径 vs log(N)（L0=256/512/1024/2048） | **R²=1.0000**（直径=14,16,18,20） | 直径∝log(N)＝双曲/小世界（平坦 N^(1/d) でない） |
| 動径距離 (0,0)→(層l,0) | 0,1,2,…,9 | オクターブが離れるほど動径方向に離れる（夢3）✓ |

### Part B — 螺旋＝繰り返し＋微小な非対称
| 量 | 測定値 | 解釈 |
|---|---|---|
| ε=0（対称） | 閉合変位 **3.1e-15**、半径一定 | 円＝戻って潰れる ✓ |
| ε=0.02（非対称） | d(log r)/dstep=**0.019803**＝log(1+ε) | 螺旋＝進む（log半径が線形） ✓ |
| 一周の進み | 各周 +1.2674（一定） | 段が等間隔＝積まれる ✓ |
| オクターブ橋（(1+ε)^N=2） | log₂半径/周 = 0,1,2,3,4,5 | **一周＝一オクターブ**（+1ちょうど） ✓ |

### Part C — 全帯を束ねる（ホログラフィー的層構造）
| 量 | 測定値 | 解釈 |
|---|---|---|
| 連結性 | 連結（1023/1023） | 一つの幾何に束ねられる ✓ |
| 境界間距離 (0,0)→(0,j) | 1,2,4,6,8,10,12,14,16（j=1..256） | **~log(分離)** R²=0.9972 vs 線形 0.6986 |
| 動径（束ねる方向） | 512→256→…→1 | 半減＝ホログラフィー動径＝繰り込み |

境界間が log(分離)＝測地線がバルクに潜る＝**Ryu-Takayanagi 的**（構造として）。

### Part D — 繰り込み × ホログラフィー（構造的対応のみ）
各折り畳み層＝自由度半減（RG一段）＝測った双曲幾何の動径シェル一枚。境界距離
~log は構造として RT。**定量証明ではなく構造対応（interpretive/speculative）**。
平坦な実空間はこれだけでは出ない（2-cell/「第三」が要る）。

## 床（隠さない）
1. **オクターブ階層は手作り**。2分木は本来双曲なので「双曲が創発した」は実質
   「双曲な階層を作って測った」＝再導出であって驚きではない。
2. **ε（51:49 の非対称）は手入れ**。測ったのは「非対称→螺旋→進む＋積む」で
   あって、非対称そのものの創発（0≠無の根）ではない。それは (B) の深い床（未着手）。
3. **MERA/AdS-CFT の構造再現であり新しい物理ではない**（analogy）。
4. **平坦な実空間（3次元）はこれだけでは出ない**。2-cell/「第三」が要る（既知の床）。
   折り畳み方向はホログラフィー動径の handle であって平坦空間の handle ではない。
5. 仮説の一点（螺旋の段＝オクターブ＝周波数帯＝折り畳み層＝繰り込みスケール＝
   動径次元）は、Part A〜C で **構造として整合**を測れたが、宇宙論的な意味づけは
   **speculative**。measured とは混ぜない。
