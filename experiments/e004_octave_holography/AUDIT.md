## 測定結果（`result.json`、L0=512）

### Part A — 折り畳み階層は双曲か（**平坦対照で判別を検証**）

**判別する証拠（同じ測定法を既知の平坦2D格子に通して検証）：**
| 量 | 階層 | 平坦対照(2D) | 解釈 |
|---|---|---|---|
| ボール成長 R²（指数 / べき） | 0.9825 / 0.9616 → **指数勝ち** | 0.9463 / 0.9965 → **べき勝ち** | 同じ法が階層=双曲・平坦=平坦と正しく分類 |
| 次元スロープ 1/d_eff（log直径 vs logN） | **0.171（≈0）** | **0.503（≈1/2）** | 階層=双曲（無限次元的）、平坦=2次元 |
| 判別成立 | — | — | **method_discriminates = True** ✓ |

→ R² の指数 vs べき比較は **異なる x 変換上の比較で形式的なモデル選択ではない**（床参照）。
そこで (i) 変換に依らないスカラー「次元スロープ」と (ii) **平坦対照**で判別を裏づけた。
この2点が**判別する測定**＝双曲分類は当てはめ作為ではない。

## 床（隠さない）
1. **オクターブ階層は手作り**。2分木は本来双曲なので「双曲が創発した」は実質
   「双曲な階層を作って測った」＝再導出であって驚きではない。
2. **ε（51:49 の非対称）は手入れ**。測ったのは「非対称→螺旋→進む＋積む」で
   あって、非対称そのものの創発（0≠無の根）ではない。それは (B) の深い床（未着手）。
3. **MERA/AdS-CFT の構造再現であり新しい物理ではない**（analogy）。
4. **平坦な実空間（3次元）はこれだけでは出ない**。2-cell/「第三」が要る（既知の床）。
   折り畳み方向はホログラフィー動径の handle であって平坦空間の handle ではない。
5. 仮説の一点（螺旋の段＝オクターブ＝周波数帯＝折り畳み層＝繰り込みスケール＝
   動径次元）は、Part A〜C で **構造として整合**を測れたが、宇宙論的な意味づけは
   **speculative**。measured とは混ぜない。
6. **統計の床（レビュー反映）**：ボール成長の「指数 vs べき」R² 比較は *異なる x
   変換上の比較*で、形式的なモデル選択ではない（差 0.02 は小さい）。これを補うため
   (i) 変換に依らない**次元スロープ 1/d_eff**（階層0.17 / 平坦0.50）と (ii) **平坦
   対照**を導入し、測定法が幾何を判別することを示した。直径∝log(N)・境界∝log・
   動径=層 は **2分木の準トートロジー**で、それ自体は証拠でなく構成の帰結。
   ただし「純粋な粗視化結線だけで log直径が出て、平坦格子とは明確に分離する」こと
   自体は **本物の measured-structural 結果**（手で曲率は入れていない）。
7. **オクターブ橋（(1+ε)^N=2 で一周=一オクターブ）は構成上の恒等式**（ε を選んで
   そうなる）。BFS/数値で確認した「構成の検証」であって、創発の発見ではない。
